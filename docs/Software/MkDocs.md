# MkDocs

## MkDocs Install

---

```sh
pip install mkdocs
# or brew install mkdocs

pip install mkdocs-material
```

## MkDocs Config

---

```sh
# Creating Site
mkdocs new .

# Previewing as you write
mkdocs serve

# Building Site
mkdocs build
```
